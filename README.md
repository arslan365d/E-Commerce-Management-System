# E-Commerce-Management-System
This project is a console-based E-Commerce Management System developed in C++ using Object-Oriented Programming (OOP) principles. It simulates the core functionalities of an online shopping platform, allowing users to browse products, manage carts, and process orders efficiently.

The system is designed with modular and reusable classes, demonstrating key OOP concepts such as encapsulation, inheritance, polymorphism, and abstraction.
